Mac OS

To install python & cookiecutter it requires brew to be installed.
if you already have installed brew then you can skip first step.

1. Install Brew

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

after installation done, it asks to sets some enivorment variables.

2. Python

brew install python

3. Cookiecutter

brew install cookiecutter

Ubuntu

1. Python

sudo apt install python3.8

2. Cookiecutter

python3 -m pip install --user cookiecutter
